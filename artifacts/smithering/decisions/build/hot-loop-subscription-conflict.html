Hot-loop subscription probe conflict

BUILD   probe-hot-loop-llm-subscription  ·  2026-06-15

Decision

Report P-LLM/A-LLM-SUB as a red probe verdict and failed gate rows. The host Codex subscription route and the Claude CLI route both return schema-compatible answers through the host subscription transport, but the measured CLI round trip is far outside the 100 ms hot-loop decision budget. The live cost gate is also not proven because the host subscription CLIs expose no per-call metering to this probe. This is surfaced to the gate as the binding PRD §6 conflict requested by the ticket.

Alternatives

Add a Cerebras base URL fallback. Rejected. The ticket explicitly forbids building around the subscription-route gate.

Record the red verdict and block dependents. Chosen. It preserves PRD §6 honesty and leaves the gate to resolve the product conflict.
